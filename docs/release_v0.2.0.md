### Summary
- Introduces a second, opt-in method for span-based Tx measurement (T10/T20/T30/T60).

- Updates documentation and docstrings for the expanded API surface.

### Highlights
- Interactive span selector for analyst-driven Tx measurements.
- Reports slope (dB/s), linearity metrics, and optional zeta when fn is provided.
- Public API entry point for launching the Tx span UI.
